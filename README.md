# 🌐 Modern Developer Portfolio

A clean, responsive, and interactive personal portfolio website built to showcase my projects, skills, and technical experience.

🔗 **[Live Demo](https://liampai-dev.vercel.app/)**

---

## 🌟 Key Features

- **Native Animations:** Uses the browser's native `IntersectionObserver` API for smooth, performance-first section transitions without heavy external libraries.
- **IDE Mockup Component:** Interactive code editor section displaying developer profile data with responsive hover effects.
- **Tailwind v4 Styling:** Modern UI architecture built utilizing the latest Tailwind CSS utilities.
- **Fully Responsive:** Optimised layout that adapts seamlessly across mobile, tablet, and desktop viewports.

## 🛠️ Tech Stack

- **Framework:** React
- **Language:** TypeScript
- **Styling:** Tailwind CSS (v4)
- **Deployment:** Vercel

---

## ⚙️ Local Setup & Installation

Follow these steps to run the project locally on your machine:

```bash
# 1. Clone the repository
git clone [https://github.com/paingzinyemaung/your-repo-name.git](https://github.com/paingzinyemaung/your-repo-name.git)
cd your-repo-name

# 2. Install dependencies
npm install

# 3. Run the development server
npm run dev
```

Open **http://localhost:5173** inside your browser to view the application.
